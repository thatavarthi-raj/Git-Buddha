# Git-Buddha
### The Ultimate `.gitkeep` Manager — With Zen Mode

> **"Before enlightenment: empty folders disappear.  
> After enlightenment: they are preserved with wisdom."**

The smartest, cleanest, and most peaceful way to keep empty directories in Git.

```bash
git-buddha --zen
```

## Features

- Zero dependencies (pure Python stdlib)
- Super fast & memory efficient
- Zen mode (no output, just peace)
- Intelligent `.gitkeep` with purpose & timestamp
- Works everywhere (Windows, macOS, Linux)

## Install

```bash
curl -L https://raw.githubusercontent.com/totalbrain/Git-Buddha/main/git-buddha.py -o git-buddha.py
python git-buddha.py --zen
```

## Usage

```bash
git-buddha                 # Current directory
git-buddha src/ tests/      # Multiple paths
git-buddha --zen            # Pure enlightenment
```
