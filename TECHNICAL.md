# git-buddha — Technical Documentation

Clean, fast, zero-dependency Python tool for preserving empty directories in Git.

- Pure stdlib · No pip install needed
- Generator-based scanning (memory efficient)
- Cross-platform (Windows/macOS/Linux)
- Idempotent & safe
- Zen mode ready
